# UI & API Automation Framework

Framework ini dibuat untuk pengujian UI & API website secara otomatis menggunakan:

- Java
- Gradle
- Selenium WebDriver
- Cucumber (BDD)
- Page Object Model

## Struktur Proyek
- pages: Page Object Model
- steps: Step Definitions
- features: Gherkin scenarios
- runner: Test Runner

## Menjalankan Test
gradle test

## Laporan
Cek laporan otomatis: build/reports/cucumber.html
